Instruções

Seja bem vindo à minha demonstração de uso desta api. Aqui faço uma simples demonstração de front-end usando html, css, js e nosso querido jQuery. Não precisavamos de nada complexo, só queremos ver a API funcionando.

Para realizar seus testes, verifique o seu ambiente em que nossa api está sendo executada, caso haja divergências corrija os dados. Logo em seguida clique em 'Autenticar' para gerar um token. Se tudo estiver ocorrido bem, você verá o token no card 'Token API'.

No card 'Feedback' lhe mostrarei mensagens de erros e sucesso que possam vim a acontecer.

No card 'Meu produto' você deverá descrever um produto fictício que usaremos na nossa simulação de compra. Após informar os campos detalhados clique em 'Adicionar ao carrinho'. Logo em seguida seu produto será listado no card 'Meu carrinho'. Lá você poderá acompanhar os produtos que você usará nesta simulação e remover caso assim deseje.

Após adicionar seus produtos fictícios, vá ao card 'Pagamento', lá você verá o valor total da nossa compra simulada. Nele você deverá informar os dados de pagamento e clicar no botão 'Realizar pagamento', acompanhe a mensagem no feedback e se tudo ocorrer bem você acabou de simular uma compra na nossa api intermediadora de pagametos.
